# 电学遮荫效应公式体系

> 背接触背结硅太阳能电池的载流子收集与IQE理论框架
## 核心概念

### 载流子收集概率 $f_c(x,z)$

| 属性 | 说明 |
|------|------|
| **物理意义** | 太阳能电池内部某一点生成的少数载流子，被后侧 p-n 结收集到的概率 |
| **数学本质** | 在各均匀掺杂区域中满足二维齐次 Helmholtz 型方程 |
| **关键作用** | 连接材料参数与 IQE 的核心桥梁 |

$$D\nabla^2 f_c - \frac{1}{\tau}f_c = 0$$
## 区域方程（核心 PDE）

### 基区 Base
$$D_{\text{Base}} \nabla^2 f_c - \frac{1}{\tau_{\text{Base}}} f_c = 0$$

### FSF 区
$$D_{\text{FSF}} \nabla^2 f_c - \frac{1}{\tau_{\text{FSF}}} f_c = 0$$

### BSF 区
$$D_{\text{BSF}} \nabla^2 f_c - \frac{1}{\tau_{\text{BSF}}} f_c = 0$$

> **物理含义**：扩散项 $D\nabla^2$ 描述载流子扩散；复合项 $\frac{1}{\tau}$ 描述体复合损失。

## 符号定义

| 符号 | 含义 | 单位 |
|------|------|------|
| $D$ | 扩散系数 | $\text{cm}^2/\text{s}$ |
| $\tau$ | 少子寿命 | s |
| $f_c$ | 收集概率 | 无量纲 |
| $(x,z)$ | 二维空间坐标 | $\mu\text{m}$ 或 cm |

## 边界条件

### 1. 发射极/结区边界
$$f_c = 1$$

> **物理意义**：少子一旦到达收集结边缘，就视为被完全收集。

### 2. 高低掺杂结界面
- **连续性**：$f_c^+ = f_c^-$
- **电流连续**：$J_n^+ = J_n^-$ 在界面两侧相等

> **物理意义**：界面两侧载流子通量不能突变。

### 3. 未扩散 gap / 表面复合边界
$$D \frac{\partial f_c}{\partial n} = S f_c$$

其中 $S$ 称为"归一化表面复合速度"。

> **物理意义**：$S$ 越大，表面越像"少子吸收槽"，收集概率越低。

### 4. 绝缘/对称边界
$$\frac{\partial f_c}{\partial n} = 0$$

> **物理意义**：无横向载流子流出。

## 光生载流子产生率

$$G(z,\lambda) = E_0(\lambda) \cdot \alpha(\lambda) \cdot e^{-\alpha(\lambda)z}$$

| 符号 | 含义 |
|------|------|
| $E_0(\lambda)$ | 入射光子通量 |
| $\alpha(\lambda)$ | 吸收系数 |

> **物理意义**：只用 Lambert–Beer 吸收，不考虑内/外反射和光陷获。
## IQE 与短路电流

### 局部/全局 IQE
$$\text{IQE}(\lambda) = \frac{1}{0.5w_{\text{pitch}}} \int_0^{0.5w_{\text{pitch}}} \int_0^{d_{\text{Base}}} \frac{G(z,\lambda)}{E_0(\lambda)} \cdot f_c(x,z) \, dz \, dx$$
> **物理意义**：把"在哪里产生的光生少子"与"这些少子被收集的概率"加权平均，得到内部量子效率。单位：无量纲。
### 短路电流密度

$$J_{sc} = q \int_{300\text{nm}}^{1200\text{nm}} \text{IQE}(\lambda) \cdot \Phi_{\text{AM1.5G}}(\lambda) \, d\lambda$$

| 符号 | 含义 | 单位 |
|------|------|------|
| $q$ | 元电荷 | C |
| $\Phi_{\text{AM1.5G}}(\lambda)$ | AM1.5G 光子通量谱 | — |
| $J_{sc}$ | 短路电流密度 | $\text{mA}/\text{cm}^2$ |

## 基础材料模型

### 迁移率模型
$$\mu = \mu_{\min} + \frac{\mu_{\max} - \mu_{\min}}{1 + \left(\frac{N_D + N_A}{N_{\text{ref}}}\right)^\delta}$$

### 带隙缩窄
$$n_{i,\text{eff}}^2 = n_i^2 e^{\Delta E_g/kT}$$

### 俄歇复合寿命
$$\tau_{\text{Auger}} = \frac{1}{C_n N_{\text{dop}}^2}$$

### SRH 复合寿命
$$\tau_{\text{SRH}} = \frac{\tau_0}{1 + N_{\text{dop}}/N_0}$$

### 总寿命
$$\frac{1}{\tau} = \frac{1}{\tau_{\text{SRH}}} + \frac{1}{\tau_{\text{Auger}}}$$

### 表面复合速度
$$S(N_{\text{dop}}) = \frac{S_0 N_{\text{dop}}}{N_{\text{ref}} + N_{\text{dop}}}$$

## 公式逻辑链

材料参数 (掺杂 N_D, N_A)
        ↓
   迁移率 μ → 扩散系数 D
        ↓
   寿命 τ (SRH + Auger)
        ↓
   f_c (求解二维 PDE + 边界条件)
        ↓
   局部 IQE → 全局 IQE
        ↓
        J_sc
### 推导步骤

1. **材料与掺杂 → 电输运参数**
   - 掺杂浓度 $N$ 通过 Masetti 模型影响迁移率 $\mu$
   - $\mu$ 进一步影响扩散系数 $D$
   - 掺杂还通过 Scharfetter/Cuevas 参数化影响 $\tau$ 与 $S$

2. **$D$ 与 $\tau$ → $f_c$**
   - 在 Base、FSF、BSF 三个区域分别求解 Helmholtz 方程

3. **$f_c$ → IQE**
   - 局部 IQE：某个横向位置 $x$ 附近的收集质量
   - 全局 IQE：对整个器件宽度与深度的加权平均

4. **IQE → $J_{sc}$**
   - 波长积分后得到短路电流密度

## 基础模型 vs 推导结果

### 基础模型（描述材料本身电学品质）
- 迁移率 $\mu(N_{\text{dop}})$
- 带隙缩窄 $\Delta E_g$
- 寿命 $\tau$
- 表面复合速度 $S$
### 中间层结果
- 载流子收集概率 $f_c$
  - 电学遮荫效应的直接表征量
### 最终可观测结果
- 局部 IQE
- 全局 IQE
- $J_{sc}$
## 关键优化结论

> 对背接触背结电池，性能由 **几何尺寸 + 掺杂浓度 + 表面复合速度** 共同决定。

### 核心优化方向

| 方向 | 效果 |
|------|------|
| 降低 gap 的表面复合速度 $S$ | 提升 $f_c$ 的关键 |
| 缩小或消除未扩散 gap | 显著减弱电学遮荫 |
| 降低基区掺杂浓度 | 减小高低结带来的有效 SRV，提高收集概率 |
| 减小非收集基区宽度 | 缩短少子横向传输距离 |
| 优化 BSF/FSF 掺杂 | 在低接触电阻与低复合之间折中 |

> **电学遮荫的本质**：背面金属与非收集区几何引起的少子收集损失，而不是单纯光被遮住。
## 一句话总结

> 低基区掺杂、窄非收集区、低 gap SRV、优化 BSF/FSF 是抑制电学遮荫、提升 $J_{sc}$ 的核心方向。
